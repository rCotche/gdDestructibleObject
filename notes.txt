press N > scale > x y z value = 0.005

en haut à gauche (au niveau de la liste deroulante object mode) select menu object > Apply > all transforms

edit mode, press tab > hover la bougie (tout ce qui compose, la flamme la meche, crire) dans le pumpkin, press l > press x > delete vertices

press tab, switch back to object mode

en haut à droite, select material preview, ref image material_preview.png > en bas à droite (inpector) select tab material >
Puis en haut à droite, click sur le + > puis click sur le bouton new > pour créer un new material, ref image creat_material.png

select surface (par default déjà open) > select base color > change it to black > click dans le input où il ya la valeur en hex > press enter to confirm

puis select inner material > click assign
